# Small projects

## 1. Group Anagrams (groupAnagrams.py)
Given an array of words, the script returns a 2D array of the words grouped into anagrams.

## Example

**Input:**
```python
["listen", "silent", "hello", "enlist", "world"]
```
**Output:**
```python
[["listen", "silent", "enlist"], ["hello"], ["world"]]
